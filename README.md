# is-reachable-cli

> Check if hostnames are reachable or not

## Install

```
> npm install is-reachable-cli --global
```

## Usage

```
> is-reachable --help

	Example
		> is-reachable beatfreaker.github.io
		✔ ︎Reachable
```
